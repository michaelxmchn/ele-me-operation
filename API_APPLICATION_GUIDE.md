# 饿了么开放平台 API 申请指南

> 申请时间: 2026-02-12

---

## 📋 申请流程

### 1. 访问开放平台
```
https://open.ele.me
```

### 2. 注册开发者账号
- 使用企业邮箱注册
- 绑定手机号
- 设置密码

### 3. 创建应用
1. 登录控制台: https://console.ele.me
2. 点击「创建应用」
3. 填写信息:
   - 应用名称: 智能运营助手
   - 应用类型: 商家应用
   - 应用描述: 自动处理订单、调整价格、管理推广

### 4. 提交资质
需要上传:
- [ ] 营业执照扫描件
- [ ] 法人身份证正反面
- [ ] 店铺授权书（如非法人）
- [ ] ICP备案号（如有网站）

### 5. 等待审核
- 审核时间: 3-7个工作日
- 审核结果会发送邮件

### 6. 获取密钥
审核通过后获取:
- `AppID`
- `AppSecret`
